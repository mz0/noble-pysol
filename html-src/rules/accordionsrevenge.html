Accordion's Revenge

One deck type. 1 deck. No redeal.

Object

Remove all the cards except the one card declared at the start.

Quick Description

Like Accordion, but at the start of the game, a single card is randomly declared. This card must be the last remaining card in order to win the game.

Notes

Accordion's Revenge is unwinnable if the first or second card is declared. As such, PySol will reselect the declared card if it is chosen.

The concept of Accordion's Revenge was invented by Mark Masten.
