Four Packs

Two-Deck game type. 2 decks. 1 redeal.

Object

Move all cards to the tableau, in four columns with two complete sequences each from ace to king, from ace to king, then wrapping around, and up to king again.

Quick Description

Like Eight Packs, but the sequences can wrap from king to ace. In order to win, the cards need to be arranged in four columns, each wrapping around and containing two full sequences.
