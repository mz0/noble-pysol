Exit

Pairing type. 1 deck. No redeals.

Object

Move all cards to the single foundation.

Rules

Cards are dealt in ten columns of five cards, plus one column of two.

Pairs of exposed number cards can be removed if they total up to 11. Jacks are removed in pairs. A queen can be removed if it is paired off with a king, but not if they are of the same suit. Removed cards are moved to a single foundation, and the game is won if all cards are paired off.
