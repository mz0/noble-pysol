Lucky Thirteen

One-Deck game type. 1 deck. No redeal.

Object

Move all cards to the foundations.

Rules

Cards are dealt to thirteen piles of four cards each, arranged so only the top card is visible. Tableau piles are built down by rank, regardless of suit. Only single cards can be moved between piles, and empty piles cannot be filled. The foundations are built up by same suit from ace to king. The game is won if all cards are moved to the foundations.
